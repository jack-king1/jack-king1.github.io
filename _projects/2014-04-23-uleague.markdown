#### What is uleague?

Using the Riot Games API this website gets live summoner data on players searched (EU West Servers) and displays it back to them. It currently gives details on the players recently played matches and links to other players, items built, KDA, Creep Score etc. It also has functionality to display all champions lore with spell explanations - An additional feature i would like to add is displaying active live game data to the user before the game starts.
